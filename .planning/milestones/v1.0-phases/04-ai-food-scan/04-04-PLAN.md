    Create backend/src/scripts/seed-foods.ts following the exact pattern of seed-exercises.ts:

    1. Import dotenv/config (loads backend/.env for MONGODB_URI)
    2. Import mongoose
    3. Import FoodItem from '../models/FoodItem'
    4. Import/require the JSON data: `import foodData from './data/vietnamese-foods.json'` (with `assert { type: 'json' }` or use `fs.readFileSync` + JSON.parse — use whichever matches seed-exercises.ts pattern)

    Script body:
    - Connect to mongoose using process.env.MONGODB_URI
    - Log "Connected to MongoDB"
    - count = await FoodItem.countDocuments()
    - Log `Found ${count} existing FoodItem documents`
    - if count >= 50: log "FoodItem collection already seeded (>= 50 items). Skipping." and exit 0
    - items = foodData.map(item => ({ ...item, source: item.source || 'manual' }))
    - await FoodItem.insertMany(items, { ordered: false })
    - Log `Seeded ${items.length} Vietnamese food items`
    - disconnect, exit 0

    Wrap in try/catch: on error, console.error + disconnect + process.exit(1)

    Update backend/package.json scripts section:
    Add: `"seed:foods": "tsx src/scripts/seed-foods.ts"`

    This follows the same pattern as `"seed": "tsx src/scripts/seed-exercises.ts"`.
  </action>
